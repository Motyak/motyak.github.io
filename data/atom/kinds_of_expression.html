Here are all kinds of expression we can write in Monlang:


primitive expressions:

fncall()
fncall(arg)
fncall(arg1, arg2)
():{'someval}
(x):{x}
(_, _):{}
arr[#nth]
arr[#from..to]
arr[#from.. map(times2)
5 in 1 .. 10 || ERR("somethin' wrong")


block expression:

- built from statements
- makes an expression grow vertically
- as within a Program, statements are evaluated
  from top to bottom
- the trailing statement, if producing a value,
  is used to return as the final result
  of the all block expression

{
    var a 1 + 2
    var b 3
    a * b
}
